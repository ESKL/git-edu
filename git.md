# git lesson
 **Git** - распределенная система контроля версий  
 **GitHub/GitLab** - это хранилища для удаленных репозиториев Git с интрефейсом для CI|CD

## git commands 
*Hастройка:*  
**git config --global user.name** - *установка имени пользователя*  
**git config --global user.email** - *установка почты пользователя*  

*Инициализация:*  
**git init** - *инициализация репозитория*    
**git clone <repo-url>** - *клонирование репозитория

*Добавление файлов индекс*  
**git add .** -*добавить файлы в репозиторий, где . указатель на текущую директорию*    
**git add file.java** -*добавить конкретный файл в репозиторий*  
__git add *.java__ -*добавить все файлы расширения .java в репозиторий*  
**git add -f file.log** -*добавить игнорируемый файл в репозиторий*  

*Создания коммита*   
**git commit -m "Message"** - *зафиксировать текущее состояние*  
**git commit -am "Message"** - *добавление ранее отслеживаемых файлов и зафиксировать их текущее состояние*   

**git status** - *информация о текущем состоянии репозитория*  
**git log** - *вывести историю фиксаций (коммитов)*  
**git checkout** - *изменить текущую версию (перемещение между коммитами)*
